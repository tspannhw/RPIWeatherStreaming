# RPIWeatherStreaming
Similar to https://github.com/tspannhw/RPIThermalStreaming for another machine


### To Run

````

python3 weather_main.py --batch-size 100 --interval 10.0 --fast 

<or>

python3 weather_main.py --batch-size 1000 --interval 100.0 --fast

<or>

python3 weather_main.py --batch-size 100 --interval 10.0 --fast --verbose


````
